# 🌐 Portfolio Website

A modern, responsive Portfolio Website built using **HTML5, CSS3, and JavaScript**. This project demonstrates front-end web development fundamentals without using any frameworks.

## 🚀 Features

- Responsive Navigation Bar
- Mobile Hamburger Menu
- Hero Section
- About Section
- Skills Section
- Projects Section
- Contact Form
- Form Validation using JavaScript
- Smooth Scrolling
- Dark Mode Toggle
- Scroll to Top Button
- Responsive Design using CSS Grid & Flexbox

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- CSS Grid
- Flexbox
- Media Queries

## 📱 Responsive Design

The website is optimized for:

- 📱 Mobile Devices
- 📟 Tablets
- 💻 Desktop Screens

## 🎯 Project Objective

The objective of this project is to create a fully responsive and user-friendly portfolio website using only HTML, CSS, and JavaScript while following modern web development practices.
